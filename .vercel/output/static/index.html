内容驱动的网站

使用 Nuxt Content 和 Markdown 轻松创建你的博客与文档站点

欢迎来到我的博客 👋

这是一个使用 Nuxt Content 驱动的博客站点。你可以使用 Markdown 编写内容，Nuxt Content 会自动将其渲染为美观的页面。

特性

  • 📝 Markdown 驱动 — 使用 Markdown 编写你的内容
  • 🎨 代码高亮 — 内置代码语法高亮
  • 📑 自动目录 — 根据标题自动生成目录
  • 🔍 全文搜索 — 支持内容搜索
  • 🚀 快速开发 — 热更新，即改即见

快速开始

在 content/ 目录下创建 .md 文件即可添加新内容。例如：

content/
├── index.md          # 首页
└── blog/
    ├── first-post.md  # 博客文章
    └── second-post.md

开始探索吧！查看 博客文章 了解更多。
